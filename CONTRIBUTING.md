# 🤝 Contributing to TruthBounty API

Thank you for your interest in contributing! This guide will help you get started and ensure consistency across contributions.

---

## 📚 Overview

This project is a backend API built with:

- **NestJS (TypeScript)**
- **Prisma ORM**
- **SQLite/PostgreSQL**
- **Jest for testing**
